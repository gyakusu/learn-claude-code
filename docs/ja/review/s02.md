## 最も大事な学習ポイント（3つ）

### 1. ループは変わらない（Open-Closed原則）
s01→s02で機能が4倍に増えたのに、`agent_loop`関数の**構造は同一**。これがエージェント設計の核心です。新しい能力を足すとき、ループ（=ハーネス）を書き換える必要がない。

### 2. ディスパッチマップ = 拡張ポイント
```python
TOOL_HANDLERS = { "tool_name": handler_fn, ... }
```
`if/elif`チェーンではなく**dictで分岐**する。新ツール追加は「ハンドラ関数を書く」「dictに1行足す」「スキーマを足す」の3ステップだけ。

### 3. `safe_path()` = 最小限のセキュリティ境界
bashだけだと何でもできてしまう。専用ツール化することで、**ツールごとにセキュリティポリシーを強制**できる。`resolve()` + `is_relative_to()` の2行でパストラバーサルを防ぐ、シンプルだが効果的なパターンです。

---

一言でまとめると: **「エージェントの能力拡張とは、ループを変えることではなく、ディスパッチマップにツールを追加すること」** — これがs02の本質的なメッセージです。
